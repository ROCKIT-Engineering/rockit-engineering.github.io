Detailed Description

template<typename T>
class TaskWrapper_t< T >

Abstract base class for wrapping FreeRTOS tasks with start/stop control and lifecycle hooks.

Usage:

  • Inherit from TaskWrapper and implement the run() method with the task's main logic.
  • Optionally override onSuspend() to perform actions right before the task is suspended.
Parameters
nameThe name of the task (for debugging and identification).
priorityThe priority of the task (default: tskIDLE_PRIORITY + 1).
stackDepthThe stack size for the task (default: 128).
